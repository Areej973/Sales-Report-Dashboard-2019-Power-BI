This project presents an interactive Sales Report Dashboard for the year 2019, developed using Power BI. The dashboard provides a clear overview of sales performance and key business metrics through dynamic and interactive visualizations.

## Dashboard Highlights

Overview of total sales, COGS, and gross income

Sales analysis by product line

Sales distribution by branch and city

Monthly sales and gross income trends

Sales breakdown by gender

Quantity sold by product line

Interactive filters for monthly analysis

## Tools Used

Power BI

DAX Measures

Data Modeling

Data Visualization Techniques

## Objective

The objective of this dashboard is to analyze 2019 sales data and convert raw data into meaningful insights that support business performance evaluation and data-driven decision-making.
